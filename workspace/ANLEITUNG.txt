Kommandozeile:
1. cd zum workspace ordner
2. npm start : startet den lokalen server, es wird dann angezeigt unter welchem http-pfad ihr die seite im browser öffnen könnt
3. neues fenster öffnen in der kommandozeile
4. wieder cd zum workspace ordner
5. npm run webpack : compiliert den inhalt der main.js und legt den compilierten code in build ab
